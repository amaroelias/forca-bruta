2. COMPILAR CADA CLASSE COM O COMANDO: `JAVAC NOMEDACLASSE.java`
3. ESCOLHER O NUMERO DA PERMUTAÇÃO E USAR O COMANDO: `JAVA NOMEDACLASSE NÚMEROPERMUTAÇÃO`
